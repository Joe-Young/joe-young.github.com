Hadoop

  1. apt-get install openssh-server

  2. 安装JDK
    vim /etc/profile

    source /etc/profile

  3. 配置/etc/hosts。
    配置ssh：启动服务/etc/init.d/ssh start （service sshd start，测试ssh localhost，本地测试）。
    ufw disable( status) 关闭防火墙。
    reboot

  4. ssh-keygen -t rsa
    cd .ssh/ && cat id_rsa.pub >> authorized_keys

  5. 配置文件/hadoop/conf
    存储目录
    /home/l/work/hadoopdir
    配置hadoop-env.sh
    java目录
    export JAVA_HOME=/home/l/work/envoriment/jdk

  1. 配置：core-site.xml、hdfs-site.xml、mapred-site.xml
